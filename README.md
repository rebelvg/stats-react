# stats-react

## Overview
Frontend counterpart for [stats-node](https://github.com/rebelvg/stats-node) server.

### Requirements
- Node.js >= 8
- Yarn

## Usage

### Run
```
yarn install
yarn run start
```

### Build
`yarn run build`

### Libraries
- React
- Redux
- React-Router
- Styled-Components
- Decorators
